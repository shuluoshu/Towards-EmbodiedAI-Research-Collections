This report follows a strict filtering and analysis format:

- prioritizes first-party sources such as papers, official technical reports, official blogs, project pages, GitHub repositories, model cards, and benchmark pages;
- separates confirmed works from ambiguous or weakly sourced items;
- includes trend summaries, work tables, taxonomy maps, organization timelines, dependency relations, idea clusters, per-work technical cards, and core mechanism diagrams;
- excludes PDF binaries from the repository to keep it lightweight. Official PDF links are preserved inside the report.

## Scope

The current report covers key works around:

- Vision-Language-Action models
- World Models and World Action Models
- robot foundation models
- diffusion / flow / action-token policies
- action representation and cross-embodiment transfer
- data engines, simulation, benchmarks, and tactile / multimodal embodied control

The HTML files are self-contained for local reading. On GitHub, the Markdown report is the most readable default entry; the HTML files can be downloaded or served through GitHub Pages if enabled later.
